# Hospital Emergency Room Dashboard (Power-Bi Project)
This project presents an interactive Emergency Room (ER) Dashboard built using Power BI to analyze patient flow, waiting time, and overall hospital performance. The dashboard helps in understanding operational efficiency and identifying areas for improvement in emergency healthcare services.

## 📌 Project Overview
This project presents an interactive Emergency Room (ER) Dashboard built using Power BI to analyze patient flow, waiting time, and overall hospital performance.

The dashboard helps in understanding operational efficiency and identifying areas for improvement in emergency healthcare services.

## 🎯 Objectives
•	Monitor patient volume and 

•	Analyze average waiting time

•	Track admission vs non-admission rates

•	Evaluate department-wise referrals

•	Identify peak hours and workload patterns


## 📊 Dashboard Features
•	⏱️ Average Waiting Time Analysis

•	👨‍⚕‍ Total Patients Count

•	🏥 Admission vs Non-Admission Split

•	📍 Department Referrals Breakdown

•	📅 Time-Based Trends (Hourly/Daily)

•	📈 Interactive filters for better insights

## 📈 Key Insights

•	Peak patient inflow occurs during specific hours, leading to increased waiting times

•	Certain departments receive higher referrals, indicating workload imbalance

•	A significant portion of patients are non-admitted, suggesting scope for outpatient optimization

•	Waiting time increases during high patient volume periods

## ✅ Conclusion
The Hospital Emergency Room Dashboard provides a comprehensive view of patient flow, waiting times, and departmental workload, enabling better operational decision-making.
The analysis highlights that patient inflow is unevenly distributed across time periods, leading to peak-hour congestion and increased waiting times. Departments with higher referral rates experience greater pressure, indicating a need for improved resource allocation and staffing strategies.

Additionally, the presence of a significant number of non-admitted patients suggests opportunities to optimize triage processes and enhance outpatient service efficiency.
Overall, this dashboard demonstrates how data visualization can help hospitals:

•	Reduce patient waiting time

•	Optimize staff and resource allocation

•	Improve emergency response efficiency

•	Enhance overall patient experience

By leveraging these insights, healthcare institutions can move toward more data-driven, efficient, and patient-centric operations.
